# Items and Sizes
## Bulk Conversions
| Creature Size | Bulk Limit | Treats as Light | Treats as Negligible |
| ------------- | ---------- | --------------- | -------------------- |
| Tiny          | Half       | -               | none                 |
| Small or Med  | Standard   | L               | -                    |
| Large         | x2         | 1 Bulk          | L                    |
| Huge          | x4         | 2 Bulk          | 1 Bulk               |
| Gargantuan    | x8         | 4 Bulk          | 2 Bulk               |
